# Michael Chen

AI software engineer at Epoch building agentic developer tools, evaluation harnesses, and multi-agent simulation systems. CS student at UC Riverside, graduating June 2026, with a minor in Entrepreneurship & Strategy.

My work sits at the intersection of autonomous coding agents, AI infrastructure, and product engineering. Recently, I have worked on Zerg, an agent framework for generating and maintaining codebases; ZTC, an agentic terminal with parallel tool execution and isolated batch runs; continual-validation systems for AI behavior regression testing; and forward-deployed engineering for production AI applications.

Previously, I built automated test systems at Mobivolt for a DOD-funded fuel-cell program, integrating 250+ sensors across 5,000+ hours of testing. I also founded AI@UCR, UCR's first official AI organization, scaling it to 50+ active members and running technical workshops on applied AI systems.

## Featured work
